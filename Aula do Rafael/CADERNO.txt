COMANDOS:

	Bandeirinha + E --> Abre Explorer do Windows

	CTRL + SHIFT + N --> Cria uma pasta 

	ALT GR + TECLA COM TRÊS CÓDIGOS --> Vai o código do meio

	ALT + TAB --> Fica migrando entre telas diferentes

	CTRL + S --> Salva o arquivo

	SHIFT + END --> Seleciona de onde o cursor tá, até o final da linha

	SHIFT + HOME--> Seleciona de onde o cursor tá, até o início da linha

	SHIFT + SETA PRA BAIXO --> Seleciona de onde o cursor tá para outras linhas a baixo

	CTRL + X --> Recorta a coisa selecionada

	ALT + TECLA SELECIONADA NO VS CODE --> Seleciona mais coisa 

	DELETE --> Apaga pra direita

	BACKSPACE --> Apaga pra esquerda

	ALT + Z --> Quebra de Linha no VS CODE

	CLIQUE DO SCROLL DO MOUSE --> Abre uma aba externa

	F2 - Renomear arquivo
